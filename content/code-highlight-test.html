Python Example

Here's a simple Python function to calculate factorial, wrapped in `pre` and `code` tags with a language class:


def factorial(n):
    """
    This function calculates the factorial of a non-negative integer.
    """
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

# Test the function
num = 5
print(f"The factorial of {num} is {factorial(num)}")

JavaScript Example (Auto-detection)

This JavaScript snippet demonstrates an asynchronous operation. Notice there's no `language-` class, so `highlight.js` should auto-detect:


async function fetchData(url) {
    try {
        const response = await fetch(url);
        if (!response.ok) {
            throw new Error(`HTTP error! Status: ${response.status}`);
        }
        const data = await response.json();
        return data;
    } catch (error) {
        console.error("Could not fetch data:", error);
        return null;
    }
}

fetchData('https://api.example.com/data')
    .then(data => console.log(data))
    .catch(err => console.error(err));
